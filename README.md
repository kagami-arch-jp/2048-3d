# 2048 3D

3D 2048 puzzle game built with Three.js. Swipe or use arrow keys to merge tiles. Tilt and orbit the board with mouse/touch.

**Live Demo:** https://2048-3d.pages.dev/

## Controls

| | Desktop | Mobile |
|---|---|---|
| Move tiles | Arrow keys / WASD | 1-finger swipe |
| Orbit camera | Click & drag | 2-finger drag |
| Reset camera | ◉ button | ◉ button |

## Tech

- [Three.js](https://threejs.org/) — WebGL 3D rendering
- Cloudflare Pages — hosting & deployment
